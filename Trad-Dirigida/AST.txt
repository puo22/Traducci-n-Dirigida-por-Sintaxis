Ejemplo de AST decorado para: a + b * c

Representación del árbol con atributos 

E
├─ E₁ → trad = "a"
├─ + 
└─ T → trad = "mul(b, c)"
     ├─ T₁ → trad = "b"
     ├─ *
     └─ F → trad = "c"

Resultado final:
E.trad = "suma(a, mul(b, c))"

Este AST se genera mediante el ETDS y se imprime como resultado de la traducción.
